## Passo 1: Criar Repositório no GitHub
1. Acesse: https://github.com/new
2. Nome do repositório: `GraphNexus`
3. Descrição: "AI-powered knowledge graph system with Neo4j"
4. Clique em "Create repository"

## Passo 2: Copiar URL HTTPS
Após criar, copie a URL HTTPS que aparecerá (ex: https://github.com/seu-usuario/GraphNexus.git)

## Passo 3: Executar no Terminal
```bash
# Substitua pela URL do seu repositório
git remote add origin https://github.com/SEU-USUARIO/GraphNexus.git
git branch -M main
git push -u origin main
```

## Pronto!
Seu repositório estará em: https://github.com/SEU-USUARIO/GraphNexus
